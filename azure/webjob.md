## Want to stop Web Job

1.ポータルからApp Serviceを停止させた後に [Resource Explorer](https://resources.azure.com/) で
Searchの隣の Directory を選択する。

2.SearchでWeb Job名を検索する

3.該当のサイトの scmSiteAlsoStopped を true に設定する。
